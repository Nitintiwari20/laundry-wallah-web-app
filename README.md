Project Overview

The website is designed to look simple, clean, and easy to use.
Users can navigate through sections like Home, Services, About Us, and Contact Us.
There is also a booking system where users can add services and see the total amount in real-time.

1️⃣ Navigation Bar
Responsive navbar
Logo on the left
Navigation links (Home, Services, About, Contact)
User name section on the right

2️⃣ Hero Section
Heading, short description
“Book a Service Today” button → scrolls to booking section
Laundry-related image on the right

3️⃣ Service Overview Section
Shows basic achievements:
15+ Services
2400+ Happy Customers
2+ Years of Experience

4️⃣ Booking Section
This is the main functional part of the website:
Left Side – Service List
List of services with names and prices
"Add Item" and "Remove Now" buttons
User can add items to the cart
Right Side – Cart + Booking Form
Displays added items
Real-time total update
Booking form: Full Name, Email, Phone
“Book Now” triggers EmailJS
Shows a thank-you message after booking

5️⃣ EmailJS Integration
When a user clicks on Book Now:
A confirmation email is sent to my email
Contains order details and total amount
